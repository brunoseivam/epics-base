EPICS Base Release 3.16.0

This version of EPICS Base has not been released yet.

Changes between 3.15.x and 3.16.0

Update antelope and e_flex

Pull in newer versions of the lexer and parser generator tools to allow generation of re-entrant parsers. Also, newer byacc supports bison destructor extension to simplify cleanup of parser stack.

Update e_flex from 2.3 (circa 1993) to 2.5.29 (circa 2003). This is the most recent version before flex began depending on M4 at runtime.

Update antelope (aka. byacc) to version 20141006. Don't really know what version was in use before.

Callback subsystem API

Added a new macro callbackGetPriority(prio, callback) to the callback.h header and removed the need for dbScan.c to reach into the internals of its CALLBACK objects.
